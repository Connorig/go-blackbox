# go-blackbox

=======
go集成常用开发框架-快速开发
目前已集成：
1. iris web服务端
2. gorm 数据库持久化操作
3. redis 缓存客户端
4. MongoDB客户端
5. RabiitMq 工具类
6. RSA非对称加密工具类
7. 邮件发送服务
8. 定时任务Cronjob
9. Zaplog 日志框架
10. WebToken身份验证
