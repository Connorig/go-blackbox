# go-blackbox

=======
go集成常用开发框架-快速开发
目前已集成：
1. iris web服务端
2. gorm 数据库持久化操作
3. redis 缓存客户端
4. MongoDB 客户端
5. RabiitMq 封装工具
6. RSA 非对称加密工具
7. 邮件发送服务
8. 读取邮箱文件
9. Cronjob 定时任务
10. Zaplog 日志框架
11. WebToken 身份验证
